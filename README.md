# EmotoHI - Node.js Website

A beautiful e-commerce website for EmotoHI selling premium Ultrabe Breaks with a tropical Hawaii-inspired blue and aqua theme.

## Features

- **Home Page**: Hero section with featured products, video showcase, and why choose us
- **Products Page**: Full product catalog with 6 items
- **Gallery Page**: Image gallery showcase
- **Videos Page**: Video content grid with descriptions
- **Contact Page**: Contact form and business information
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Tropical Theme**: Blue, aqua, and teal color scheme inspired by Hawaii

## Installation & Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open in browser:
```
http://localhost:3000
```
